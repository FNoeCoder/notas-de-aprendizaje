## Formatos

| Código            | Formato de texto|
|-------------------|-----------------|
| `*Italic*`        | *Italic*        |
| `**Strong**`      | **Strong**      |
| `~~~Tachado~~~`   | ~~Tachado~~~    |

## Listas
### Simbolos que se pueden usar
| Desordenadas      | Ordenadas       |
|-------------------|-----------------|
| `* Elemento`      | `1. Elemento`   |
| `- Elemento`      |                 |
| `+ Elemento`      |                 |

### Sublistas
Agregar una tabulación al querer agregar una sublista

```
- Incio
    1. Elemento 1
    2. Elemento 2
- Desenlase
    - Elemento
    - Elemento
```
